Estrategias

¿Qué es la estrategia?


Una estrategia es un conjunto de acciones planificadas en el tiempo que se llevan a cabo para lograr un determinado fin.

Estrategia es: Es el proceso seleccionado mediante el cual se espera lograr alcanzar un estado futuro.

La estrategia es un procedimiento dispuesto para la toma de decisiones y/o accionar frente a un determinado escenario. Esto, buscando alcanzar una o varias metas previamente definidas.

Es decir, la estrategia es un plan mediante el cual se busca lograr una meta. Esto puede ser aplicado en diversos ámbitos como el militar, el empresarial, el personal, etc.

Por otro lado, el término hace referencia a un conjunto de planes a corto, mediano o largo plazo, bien elaborados que deben ser ejecutados para lograr las metas.

Aplicando estos conceptos a un proyecto de vida se podría decir que la estrategia, es un conjunto de acciones que se deben tomar con la finalidad de asegurar el logro de la meta planteada en un tiempo específico.

Tipos de metas personales:


Metas de corto plazo:

Son los acciones que están más cerca al presente, son el punto de partida para alcanzar los sueños y pueden darse en días, semanas o meses. Se inicia con actividades diarias o lo que se programe para una semana o un mes. Este tipo de metas son prioritarias e incluso tiene límite de tiempo para cumplirlas.

Metas a mediano plazo:

Son los acciones planteados para realizar en un futuro próximo, requiere disciplina, sacrificio y compromiso para lograrlos. Para cumplirlos se deben planear por tareas e inclusive escribirlos con el fin de encontrar una línea de direccionamiento efectivo.

Metas a largo plazo:

Son los metas soñados que requieren más tiempo para ser cumplidos, se cree que son a cinco o máximo diez años. Se requiere de un esfuerzo permanente, una disciplina férrea, resiliente y no perder de vista el objetivo, porque puede que se requiera modificar las acciones o inclusive las mismas metas.
